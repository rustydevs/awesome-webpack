# Awesome Webpack Starter

[![Build Status][travis-image]][travis-url]

To start development

```bash
npm run serve
```
or 

```bash
npm run serve:dev
```

To run with prod build

```bash
npm run serve:prod
```

To clean the generated files for a fresh start

```bash
npm run clean
```
